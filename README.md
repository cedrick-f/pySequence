    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-description" class="anchor" href="#description" aria-hidden="true"><span class="octicon octicon-link"></span></a>Description</h1>

<p><em><strong>pySequence</strong></em> est un logiciel <strong><a href="http://www.gnu.org/licenses/gpl.html">libre</a></strong> qui permet d'aider les professeurs de <em><strong>Sciences Industrielles de l'Ingénieur</strong></em> à :</p>

<ul class="task-list">
<li>élaborer des séquences pédagogiques</li>
<li>décrire et valider des projets</li>
</ul>

<p><em><strong>pySequence</strong></em> s'adresse particulièrement aux professeurs de <em><strong><a href="http://fr.wikipedia.org/wiki/Sciences_de_l%27ing%C3%A9nieur">Sciences Industrielles de l'Ingénieur</a></strong></em> enseignant dans les filières :</p>

<ul class="task-list">
<li>STI2D (<strong>enseignements technologiques transversaux</strong> et de <strong>spécialité</strong>)</li>
<li>S-SI (<strong>Sciences de l'ingénieur</strong> du bac S)
prochainement :</li>
<li>Technologie collège
et potentiellement <a href="/cedrick-f/pySequence/blob/wiki/Nouveau_programme.md">n'importe quel programme d'enseignement</a>.</li>
</ul>

<hr>
